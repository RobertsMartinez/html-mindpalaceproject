¿Qué es HTML?

Por sus siglas (Hyper Text Markup Lenguage), es un lenguaje de marcado que sirve para dar sentido y estructura a las páginas web. No es un lenguaje de promación, es un lenguaje de marcado que define la estructura de su contenido. Hypertexto hacer referencia a los diferentes enlaces que conectan las diferentes páginas web una con la otra. Estos enlaces son paste fundamental de la Web. HTML funcina en conjunto con otras herramientas, por ejemplo, CSS, que es utlizado para dar el diseño y la apariencia a cualquier web. A su vez, para darle funcionalidad y comportamiento usamos JavaScript

Antomía de un elemento HTML

Las partes principales del elemento son:

  1. Etiqueta de apertura: consiste en el nombre del elemento, en la imagen (p), encerrado por paréntesis angulares de apertura y de cierre. Esta establece desde que punto comienza a tener efecto el elemento selecionado.
  2. Etiqueta de cierre: similar a su contraparte de apertura, con la diferencia que contine una barra de cierre (/) antes del nombre de la etiqueta. Esta marca deja de tener efecto el elemento.
  3. Contenido: es el contenido del elemento, puede ser un párrafo, un enlace a una imagen u otra web, etc.
  4. Elemento: tanto la etiqueta de apertura, la etiqueta de cierre y el contenido, forman o equivalen al elemnto.

Los elementos pueden tener atributos. Los atributos contienen información adicional acerca del elemento, la cual no quieres que aparezca en el contenido real del elemento.

Un atributo debe tener siempre la siguiente estructura:

  1. Un espacio entre este y la línea del elemento (o del atributo previo, si el elemento ya posee uno).
  2. El nombre del atributo, seguido por un signo de igual (=).
  3. Comillas de apertura y de cierre, encerrado el valor del atributo.

Los atributos siempre se incluyen en la etiqueta de apertura de un elemento, nunca en la de cierre.

Anidar elementos

Esto significa que se pueden colorcar elementos dentro de otros elementos. A esto se le conoce como anidamiento. Los elementos deben abrirse y cerrarse ordenadamente, de forma tal que se encuentren claramente dentro o fuera el uno del otro.

Elementos vacíos

ALgunos elementos no poseen contenido, por eso son llamados elementos vacíos. Por ejemplo, el elemnto (img). Posee dos atributos, no posee etiqueta de cierre, ni contenido encerrado. Esto es porque no posee contenido que encerrar, su objetivo o próposito es desplegar una imagen en página HTML.

Anatomía de un Documento HTML

En la imagen anterior podemos listar:

  1. <"!DOCTYPE html> — el tipo de documento. Es un preámbulo requerido. Anteriormente, cuando HTML era joven (cerca de 1991/2), los tipos de documento actuaban como vínculos a un conjunto de reglas que el código HTML de la página debía seguir para ser considerado bueno, lo que podía significar la verificación automática de errores y algunas otras cosas de utilidad. Sin embargo, hoy día es simplemente un artefacto antiguo que a nadie le importa, pero que debe ser incluido para que todo funcione correctamente. Por ahora, eso es todo lo que necesitas saber.
  2. ><"html><"/html> — Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz (root element).
  3. <"head><"/head> — Este elemento actúa como un contenedor de todo aquello que quieres incluir en la página HTML que no es contenido visible por los visitantes de la página. Incluye cosas como palabras clave (keywords), una descripción de la página que quieres que aparezca en resultados de búsquedas, código CSS para dar estilo al contenido, declaraciones del juego de caracteres, etc.
  4. <"meta charset="utf-8"> —Este elemento establece el juego de caracteres que tu documento usará en utf-8, que incluye casi todos los caracteres de todos los idiomas humanos. Básicamente, puede manejar cualquier contenido de texto que puedas incluir. No hay razón para no establecerlo, y puede evitar problemas en el futuro.
  5. <"title><"/title> — Establece el título de tu página, que es el título que aparece en la pestaña o en la barra de título del navegador cuando la página es cargada, y se usa para describir la página cuando es añadida a los marcadores o como favorita.
  6. <"body><"/body> —Encierra todo el contenido que deseas mostrar a los usuarios web que visiten tu página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.

Imágenes

Esta etiqueta incrusta en el docuemnto una imagen, en la posición en que se escribe. Esto lo hace atravez de un atributo llamado surce abreviado "src" en el cual colocamos la ruta en la que se encuentra la imágen, ya sea en nuesto ordenador, alguna otra página de referencia o en los archivos de nuestra página web. También tiene el atributo "alt" en el cual colocamos una breve descripción de la imágen, esta descripción no es visible en la página web, es unicamente para que personas con discapacidades visuales, que usan lectores de pantallas puedan accesder a esa descripción. Por lo mismo la descripción debe ser clara y significativa

Marcadores de texto

Encabezados

Los elementos de encabezado permiten especificar que ciertas partes del contenido son encabezados, o subencabezados del contenido. De la misma forma que un libro tiene un título principal, y que a su vez puede tener títulos por cada capítulo individual, y subtítulos dentro de ellos, un documento HTML puede tenerlos también. HTML posee seis niveles de encabezados, van desde "h1-h6". Se debe respetar el orden de cada uno, por ejemplo, no se puede comenzar con el (h2) y luego saltar al (h4) y volver al (h1), deben ser usados en orden jerárquico.

Párrafos

Este elemento es utilizado para encerrar párrafos de texto, este es cómunmente utilizado en conjunto de el elemento (br), este último sirve para marcar el fin de una línea y dar un salto a la siguiente.

Listas

Existen dos elementos para crear listas, ya que muchos de los contenidos web, están formados por ellas. Por un lado, tenemoslas listas ordenadas (ol), y las listas que no siguen un orden numerico, las listas desordenadas (ul) unordened list, por sus siglas en inglés. Para añadir items a cualquira de los dos diferentes tipos de lista utilizamos el elemento (li) quiere decir "list item".

Las listas son bastantes versátiles, tanto que podemos agregar listas dentro de otras listas, creando así una especie de sublistas tan grande como queramos (respetando en todos los casos el orden de identación)

Vínculos

Los vínculos o enlaces son una parte mu importante, técnicamente, son la base del la Web, para implementarlos se utiliza un elemento simple (a), es la abreviatura de la palabra ancla en inglés (anchor). Este mismo elemento cuenta con un atributo llamado (href) en este se coloca ya sea el directorio de otro elemento html que hayamos creado, o el vínculo a una web que usemos como referencia.
